# Operados

## Aritméticos

- soma = +
- subtração = -
- divisão = /
- multiplicação = *
- resto = %

## Relacionais 

- < -> menor
- > -> maior
- >= -> maior igual
- <= -> menor igual
- == -> igual
- != -> diferente 

## Lógicos

- && -> and
- || -> or
- !  -> not

## Atribuição

= 
+=
-=
*=
/=

++contador -> incrementa depois executa
contador++ -> executa e depois incrementa
